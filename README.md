# IntentFlow_AI
An intent-based AI chatbot built using Python, NLTK, and PyTorch that classifies user input into intents and generates appropriate responses using a trained neural network.

It uses NLP preprocessing techniques like tokenization and lemmatization, and converts text into a bag-of-words representation for training. The model is trained using PyTorch and can be saved and loaded for reuse.

The chatbot runs in a simple terminal interface and supports multiple intents such as greeting, goodbye, programming questions, and stock-related queries.

Tech stack: Python, PyTorch, NLTK, NumPy

To run, install dependencies using pip install nltk numpy torch, download required NLTK datasets (wordnet and punkt), and run main.py.
